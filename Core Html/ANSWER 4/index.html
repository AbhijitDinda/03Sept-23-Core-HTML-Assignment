HTML Tags

  1. <html>:

    Description: Defines the root element of an HTML document. It encapsulates the entire web page content.

  2. <head>:

    Description: Contains meta-information about the document, such as the title, links to external resources, styles, and scripts. It's not displayed on the web page itself.

  3. <title>:

    Description: Sets the title of the HTML document, which appears in the browser's title bar or tab. It's the primary identifier of the webpage.

  4. <link>:

    Description: Specifies external resources like stylesheets (CSS) or icons, enabling the HTML document to connect with these external files.

  5. <style>:

    Description: Contains CSS rules for styling the document. It allows defining styles directly in the HTML file.

  6. <script>:

    Description: Embeds or references JavaScript code for client-side scripting. It enables interaction and dynamic behavior within the webpage.

  7. <body>:

    Description: Contains the visible content of the web page, including text, images, links, and other elements displayed on the browser.

  8. <header>:

    Description: Represents introductory content or a group of navigation links at the top of a page or section. It often includes headings, logos, and introductory information.

  9. <nav>:

    Description: Defines a navigation menu, typically for site navigation, providing a set of links for easy access to different sections of a website.
